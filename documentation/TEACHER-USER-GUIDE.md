## âœ… **Marking Attendance**

### **ğŸ¯ Daily Attendance Process**

#### **Step 1: Select Class and Subject**
1. Choose your assigned class from the dropdown
2. Select the subject you're teaching
3. Pick the date (defaults to today)
4. Click "Load Students" to see the class list

#### **Step 2: Mark Student Attendance**
- **Present** - Student attended the class
- **Absent** - Student was not present
- **Late** - Student arrived late to class

#### **Step 3: Bulk Actions (Optional)**
- **Mark All Present** - Quick option for full attendance
- **Mark All Absent** - For special circumstances (holidays, events)
- Use individual controls for mixed attendance

#### **Step 4: Save Attendance**
- Review your entries before saving
- Add remarks for specific students if needed
- Click "Save Attendance" to submit

### **â° Attendance Rules**

#### **Edit Window**
- **24-Hour Rule**: You can edit attendance within 24 hours of marking

- **Date Restrictions**: Cannot mark attendance for future dates

- **Duplicate Prevention**: System prevents marking attendance twice for same date
- **Assignment Verification**: Can only mark attendance for assigned class-subject combinations
- **Date Validation**: Ensures proper date format and realistic dates
